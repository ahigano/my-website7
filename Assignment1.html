2) What is the difference between HTML and XHTML?

  • HTML (HyperText Markup Language): is a standard for creating and structuring web pages. It is more forgiving of errors and allows for a more lenient syntax.
  • XHTML (eXtensible HyperText Markup Language): is a stricter and cleaner version of HTML that adheres to XML rules. It requires well-formed syntax and proper nesting of elements, making it more rigorous but also more predictable.

    7) What is the purpose of the "doctype" declaration in HTML?

    The <!DOCTYPE> declaration is used to specify the HTML version and type of document being used. It helps browsers render the page correctly by informing them about the HTML standard the document adheres to. For example, <!DOCTYPE html> indicates that the document is using HTML5. This declaration is not an HTML tag but a way to ensure consistent rendering across different browsers.


    8) What is the difference between HTML elements and HTML tags?

    HTML Tags: Tags are the basic building blocks of HTML and are used to create elements. They are enclosed in angle brackets, like <tag>. There are opening tags, such as <p>, and closing tags, such as </p>.
    HTML Elements: Elements are the complete building blocks of HTML content and include both the opening tag and the closing tag, as well as any content in between. For example, <p>This is a paragraph.</p> is a complete HTML element, with <p> as the opening tag, </p> as the closing tag, and "This is a paragraph." as the content.

    19) What is the difference between block-level and inline elements in HTML?

    Block-level elements:
    These elements take up the full width available and start on a new line. They create a "block" of content. Examples include <div>, <p>, and <h1>.
    Inline elements:
    These elements take up only as much width as necessary and do not start on a new line. They flow within the content. Examples include <span>, <a>, and <strong>.
